# 🔐 EncroLab

EncroLab is a Node.js-based encryption and decryption tool that supports multiple cryptographic algorithms (e.g., AES, DES, and others).  
It provides a step-by-step processing system to simulate how encryption and decryption work in practice.

---

## 🚧 Project Status
⚠️ **This project is currently under active development.**  
Features, structure, and implementations may change over time.

---

## ✨ Features (Planned / In Progress)
- Multi-algorithm encryption & decryption
- Step-by-step encryption & decryption process
- Support for AES, DES, and other algorithms
- REST API for integration
- Web-based interface (in progress)

---

## 🛠️ Tech Stack
- Node.js
- Express.js
- JavaScript
